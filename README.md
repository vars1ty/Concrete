# Concrete
A simple easy-to-use file format made for fun

## Example
```
// A basic comment.
root => {
    "string" => "Basic Text" | String,
    "u16" => 0 | u16,
    "i16" => 0 | i16,
    "u32" => 0 | u32,
    "i32" => 0 | i32,
    "bool" => false | bool,
}
```
